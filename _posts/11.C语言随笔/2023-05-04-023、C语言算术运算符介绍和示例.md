# 1、算术运算符

下表显示了 C 语言支持的所有算术运算符。假设变量 A 的值为
18，变量 B 的值为 5，则：

  ---------------------------------------------------------------------------
  **运算符**   **描述**                            **实例**
  ------------ ----------------------------------- --------------------------
  **+**        两个数相加                          A+B 将得到 23

  **-**        一个数减另一个数                    A-B 将得到 13

  **\***       两个数相乘                          A\*B 将得到 90

  **/**        分子除以分母                        A/B 将得到 3.6

  **%**        余数运算符，整除后的余数            B%A 将得到 3

  **++**       自增运算符，整数值增加 1            A++ 将得到 19

  **\--**      自减运算符，整数值减少 1            A\-- 将得到 17
  ---------------------------------------------------------------------------

变量的自增或自减有两种写法：

变量名++； // 表示在本次使用变量后再自增；

++变量名; // 表示在本次使用变量前自增；

变量名\--； // 表示在本次使用变量后再自增；

\--变量名; // 表示在本次使用变量前自减；
