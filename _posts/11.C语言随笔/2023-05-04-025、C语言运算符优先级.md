# 1、运算符的优先级

这篇文章可能会让您失望。

记得小学的时候就知道+-\*/四则运算符中，\*/的优先级高于+-。

在C语言中，有以下运算符：

1）算术运算符；

2）赋值运算符；

3）sizeof运算符；

4）关系运算符；

5）逻辑运算符；

6）位运算符。

运算符都有优先级，我不想介绍它们的优先级，我写了二十年的程序，也记不住各种运算符的优先级的关系，也从没打算过记住它们。

我不建议程序员记住各种运算符的优先级，而是用括号解决全部的问题，括号的优先级是最高的，括号中的表达式会优先执行，这样各种运算符的执行顺序就一目了然了。

[int d = a + (b \* c);]{.mark}

在某些场景下，用括号还可以提高程序的可读性，我是一个务实的程序员。
